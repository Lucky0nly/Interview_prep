System flow

From sign-in to scored review in one loop

01

Authenticate

Create an account or log in. The client stores a JWT and uses it for protected requests.

02

Start interview

The backend generates a fresh question set, stores the session, and returns the timer window.

03

Submit answers

Answers are evaluated by OpenAI when available, or a deterministic scoring fallback when not.

04

Track results

Dashboard APIs aggregate scores, history, and role-based performance to show progress clearly.
